## ğŸš€ Local Development Setup

### Prerequisites

- Node.js
- npm
- MongoDB

### Backend Setup

1. Clone the repository

   ```bash
   git clone [repository-url]
   ```

2. Navigate to backend directory

   ```bash
   cd backend
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Create .env file (based on .env.example)

5. Start the server
   ```bash
   npm run start
   ```
